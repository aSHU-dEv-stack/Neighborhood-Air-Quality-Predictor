Project Overview

This web application demonstrates the process of building an air quality prediction model using machine learning techniques.

The project is organized into weekly milestones:

  • Week 1: Dataset search, problem definition, and data preparation
  • Week 2: ML model selection, implementation, and evaluation
  • Week 3: Model deployment and project presentation
